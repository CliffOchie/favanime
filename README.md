# FavAnime

Show your favorite and most hated anime on your Discord profile.

## Features

- Adds an **Anime List** tab to user profiles.
- Search and add anime to your favorites (❤️) or hated (💔) list.
- Sync your lists to a server so other users can see them on your profile.

## Usage

1. Open your own profile.
2. Click the **Anime List** tab.
3. Use the **Add** button to search for anime and add them to your list.
4. Click **Sync Now** to upload your lists so others can see them.

## API Disclaimer

This plugin uses the following external APIs:

- **[Jikan API](https://jikan.moe/)** — An unofficial MyAnimeList API used to search for anime and fetch anime details. No authentication is required. Jikan's [Terms of Use](https://jikan.moe/terms) and [Privacy Policy](https://jikan.moe/terms) apply.
- **Custom sync server** — A self-hosted server used to store and retrieve users' favorite/hated anime lists by Discord user ID. No personal data beyond your Discord user ID and selected anime IDs is transmitted.
