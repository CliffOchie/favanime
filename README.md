## 📄 What is favanime?

favanime is a simple program that helps you display your favorite or least favorite anime right on your Discord profile. It uses data from MyAnimeList through JikanAPI to pull your anime list and shows it on Discord. This lets your friends and other Discord users see which anime you watch or dislike, all without complicated setup.

You do not need any coding skills. If you can download and run a file, you can use favanime.

## 🛠 How favanime Works

favanime connects to two main sources:

- **MyAnimeList**: where your anime watchlist lives.
- **Discord**: where your profile shows your chosen anime.

It uses a tool called JikanAPI to pull anime info from MyAnimeList so you don’t have to enter any data manually. The app updates your Discord profile text with your current anime preferences dynamically.

It runs quietly in the background and updates when your anime list changes or when you open the app.

## 🖥 Troubleshooting Tips

- If favanime does not show your anime on Discord:
  - Make sure you logged into Discord through the app.
  - Check your internet connection.
  - Verify your MyAnimeList username is correct.
  - Restart favanime and Discord.
- If the program won’t install:
  - Try running the installer as administrator. Right-click the file and select “Run as administrator.”
  - Check if Windows Defender or antivirus is blocking installation.
- For login issues:
  - Ensure your Discord credentials are correct.
  - Confirm Discord is not logged out elsewhere.
